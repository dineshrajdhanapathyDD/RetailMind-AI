# Product Overview

RetailMind AI is an intelligent retail decision engine built for the Indian retail market as part of the AI for Bharat initiative. It provides AI-powered inventory recommendations using Amazon Bedrock Nova Premier.

## Core Purpose

Help Indian retailers make data-driven inventory decisions through:
- Real-time inventory monitoring
- AI-generated reorder recommendations
- Actionable insights with accept/dismiss actions
- Flexible data import (CSV/JSON)

## Key Characteristics

- Localized for India (INR currency, Indian retail context)
- Serverless AWS architecture (cost-effective ~₹400-800/month for dev/test)
- Generative AI powered by Amazon Bedrock Nova Premier
- Simple, accessible interface for non-technical users
- Scalable from small shops to large chains

## Target Users

Indian retailers of all sizes who need:
- Inventory optimization
- Reduction of stockouts and overstock
- Data-driven decision making
- Competitive edge through AI insights
